# E-commerce Sales Dashboard (Power BI)

## Overview

This project is my first Power BI dashboard. The objective was to analyze e-commerce sales data and transform raw data into meaningful business insights through interactive visualizations.

## Dataset

The project uses two datasets:

* Orders
* Details

## What I Did

* Imported and transformed the data in Power BI
* Corrected and adjusted data types
* Created relationships between tables
* Built DAX measures
* Designed interactive dashboards and visualizations

## Tools Used

* Power BI Desktop
* DAX
* Data Modeling
* Data Visualization

## Dashboard Features

* Sales analysis
* Order analysis
* Product performance analysis
* Interactive filtering and exploration

## Learning Outcome

This project helped me gain hands-on experience with Power BI, data modeling, DAX, and dashboard design.
